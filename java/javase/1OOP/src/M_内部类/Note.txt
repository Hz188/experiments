内部类
    就是在一个类中，再定义一个类

格式
    public class 类名{
        修饰符 class 类名{

        }
    }

访问特点 demo1
    内部类可以直接访问外部类的成员，包括私有
    外部类要访问内部类的成员，必须创建对象

成员内部类 demo2
    按照内部类定义的位置不同，成员位置，局部位置
        成员内部类 | 局部内部类
    定义格式
        外部类名.内部类名 对象名 = 外部类对象.内部类对象
        Outer.Inner oi = new Outer().new Inner();

局部内部类 demo3
    方法中定义的一个类，外界无法直接使用，需要在方法内创建对象使用
    可以访问外部类的成员，也可访问方法内的局部变量

匿名内部类 demo4
    前提
        存在一个类或者接口，这里类可以是抽象类也可以是具体类
    格式
        new 类名 | 接口名{
            重写方法
        }
    本质是一个继承了一个类或者实现了一个接口的子类匿名对象
    是局部内部类的一种特殊形式

静态内部类 demo6
    Outer.Inner oi = new Outer.Inner();
    静态内部类不能访问外部类的非静态的变量和方法。
